Module 01

Introduction to HTML

HTML gives a page its structure — the skeleton every browser, screen reader, and search engine reads first. This module covers the document outline, semantic tags, forms, and embedded media.

Overview

What you'll learn

  • The anatomy of an HTML document — <!DOCTYPE>, <head>, and <body>
  • Semantic elements: <header>, <nav>, <main>, <section>, <footer>
  • Text, list, and table elements for structuring content
  • Building accessible forms with labelled inputs
  • Embedding images, audio, and video
Reference

Common tags & their purpose

TagPurposeExample
<h1>–<h6>Headings, in order of importance<h1>Title</h1>
<p>A paragraph of text<p>Hello</p>
<a>A hyperlink to another page<a href="...">
<img>An embedded image<img src="...">
<ul> / <ol>Unordered / ordered lists<li>Item</li>
<table>Tabular data<tr><td>
<form>Collects user input<input>
Worked example

A minimal page

Every HTML document, however large, grows from this skeleton:

<!-- minimal-page.html --> <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My Page</title> </head> <body> <h1>Hello, web!</h1> <p>This paragraph is structured with HTML.</p> </body> </html>
