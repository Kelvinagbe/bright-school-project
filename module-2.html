Module 02

CSS Styling & Layout

If HTML is the skeleton, CSS is everything else — colour, spacing, type, and the rules that decide where each element sits on the page.

Overview

What you'll learn

  • Selectors and specificity — targeting the right element
  • The box model: content, padding, border, margin
  • Flexbox for one-dimensional layout
  • CSS Grid for two-dimensional layout
  • Responsive design with media queries
