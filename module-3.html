Module 03

JavaScript Fundamentals

JavaScript turns a static page into something that responds — to clicks, to input, to time. This module covers the core language and how it talks to the DOM.

Overview

What you'll learn

  • Variables and data types — let, const, strings, numbers, booleans
  • Functions, including arrow function syntax
  • Selecting and updating the DOM
  • Listening for events — clicks, input, submit
  • Conditionals and simple control flow

Worked example

Responding to a click

// interact.js const button = document.querySelector('#greet'); button.addEventListener('click', () => { const name = document.querySelector('#name').value; alert(`Hello, ${name}!`); });
